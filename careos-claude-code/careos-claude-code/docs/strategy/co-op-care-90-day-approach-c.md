## Month 2: Pre-License Revenue (The Care Navigator Model)

### Revenue Engine
- [ ] Care Navigators begin CII/CRI assessments — $150-300 each, supervised by Dr. Emdur
- [ ] Target: 16-32 assessments/month to cover Navigator payroll (~$4,860/mo for 3 Navigators at 15 hrs/wk)
- [ ] Launch Time Bank — community members earn Care Credits for non-medical services

### What Care Navigators CAN Do (No License Needed)
- Clinical assessments (CII/CRI) under physician supervision
- Companionship visits (conversation, errands, meal sharing)
- Community resource coordination and referrals
- Time Bank facilitation
- CareOS NLP pipeline training (real workflows, non-PHI data)

### What They CANNOT Do (Until Class B License)
- Hands-on personal care (bathing, dressing, toileting, transfers)
- Medication reminders/management
- Anything CDPHE defines as "personal care services"

### Regulatory
- [ ] Criminal background checks — all staff and owners
- [ ] Secure liability insurance — $100K per occurrence / $300K aggregate (Beazley Virtual Care policy)
- [ ] Identify Administrator + Alternate Qualified Individual for CDPHE application
- [ ] Submit complete CDPHE Class B application with all documents
