Three simultaneous failures compound each other. For families: agency fees consume 40–60% of every dollar, yet turnover means a new caregiver appears constantly — destroying therapeutic relationships. For caregivers: poverty wages, zero equity, and zero voice produce 79% annual turnover — the industry's defining crisis. For the healthcare system: fragmented, undocumented care produces unplanned readmissions and zero clinical data, making outcome-based federal reimbursement structurally impossible.

The root cause: The agency model extracts 40–60% in overhead, distributes no equity, has no clinical documentation infrastructure, and is structurally excluded from the next generation of CMS value-based payment models.

Federal Moat: ACCESS Model + PACE Exclusivity

The FFS Exclusion Rule: CMS ACCESS Model participants are prohibited from billing Medicare Fee-for-Service. This bars every traditional home care agency from entering the program — permanently reserving the payment pool for qualified participants like co-op.care.
