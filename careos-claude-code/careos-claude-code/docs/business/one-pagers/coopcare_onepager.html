🔒 The Federal Moat — CMS ACCESS Model (2026)

The FFS Exclusion Rule: CMS ACCESS Model participants are prohibited from billing Medicare Fee-for-Service. This permanently bars every traditional home care agency from entering the program — reserving the payment pool exclusively for qualified participants like co-op.care.

Payment rates: $420/yr (CKM) · $360/yr (eCKM) · $180/yr (MSK + BH) per beneficiary. 10-year program (2026–2036). co-op.care also targets the PACE capitated market for near-term predictable revenue.
