## 3. COPYWRITING GEM

### Core Framing Updates

**Primary narrative: "Human Capital vs. Private Equity"**

PE firms buy agencies, cut wages to $13/hr, inflate billing to $45/hr, pocket 40–60%, flip in 3–5 years. co-op.care is structurally immune: worker ownership means no equity to extract, no company to flip, no wages to cut.

**The TrumpRx Parallel — 5 Principles:**
1. Disintermediate the extractors (TrumpRx removed PBMs; we remove PE agencies)
2. Price transparency (MFN pricing; we publish $35/hr, no hidden fees)
3. Direct access (DTC drug portal; our CII/CRI → care plan → LMN is DTC for care)
4. Outcome-aligned payments (drug pricing reflects value; CMS ACCESS pays for outcomes)
5. Data interoperability ($1M penalties for blocking; we're FHIR R4 native from birth)

**The one line:** "TrumpRx removed the middleman from prescriptions. co-op.care removes the middleman from care."
